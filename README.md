# Custom script

* Automatization script for gladiatus:
  - Quests in Pantheon
  - Expedition (incl. ruby usage)
  - Dungeon
  - Arena
  - Circus Turma
  - Smelting
  - Confirm of After fight search & other notifications (free items on new day)
  - Healing
  - Event Expedition (beta)
  - Gold storing on Auction House

Some functionality (healing/smelting/auction storage) relies heavily on the Gladiatus Crazy Addon (no special setup needed): https://gladiatus.dinodevs.com/

Installation:
---------

1. Install tampermonkey addon: http://tampermonkey.net/
2. After installation click here: https://github.com/blademarek/custom-script/raw/main/custom-script.user.js and press install.

Usage:
---------

1. Set your configuration.
2. Press Start
